# RULES

1. הריפו עוסק רק בתחום אי־ודאות, סטטיסטיקה והסתברות בסיסית.
2. קבצי מקור נשמרים בתיקיית files.
3. תיעוד AI נשמר בתיקיית docs.
4. דפי עבודה עתידיים נשמרים בתיקיית worksheets.
5. פתרונות נשמרים בתיקיית solutions.
6. אין לערבב כאן את פרויקט האחוזים הישן.
7. אין להמציא נתונים מתוך גרפים או טבלאות.
