# SOURCE PAGES MANIFEST — אי-וודאות

## Purpose

This manifest defines the required source-page inventory for the PDF `אי וודאות.pdf`.

It exists so every production worksheet page can be traced back to its original PDF page.

## Required source page directory

`assets/source-pages/`

## Required naming convention

Source pages must be stored using this exact sequence:

- `source-page-001.png`
- `source-page-002.png`
- `source-page-003.png`
- ...
- `source-page-062.png`
